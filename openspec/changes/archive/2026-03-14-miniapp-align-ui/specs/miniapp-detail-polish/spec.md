## ADDED Requirements

### Requirement: 详情页视觉完善
招投标详情页 SHALL 包含标题卡片（公告类型标签、标题、发布时间）、项目信息区、重要时间区、公告正文、原文链接按钮，布局与 ui 参考一致。

#### Scenario: 标题卡片
- **WHEN** 用户进入详情页
- **THEN** 顶部显示公告类型标签、标题、发布时间

#### Scenario: 项目信息区
- **WHEN** 详情数据包含项目信息
- **THEN** 显示项目名称、预算金额、地点、招标人、代理机构

#### Scenario: 重要时间区
- **WHEN** 详情数据包含时间信息
- **THEN** 显示报名时间、截止时间、开标时间

#### Scenario: 原文链接
- **WHEN** 用户点击底部「查看原文」按钮
- **THEN** 复制链接或打开外部链接（小程序需合规处理）
