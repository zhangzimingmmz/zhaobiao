## 精确事实源

- **接口精确真相**：后端 FastAPI 提供 `GET /openapi.json`，本地 Swagger 见 `http://localhost:8000/docs`。若需离线 YAML，可从该接口导出至 `api/openapi.yaml`（见 [api/README.md](../api/README.md)）。
- **数据库精确真相**：表结构演进在 `crawler/migrations/`，当前 schema 以按序执行这些迁移后的结果为准（见 [db/README.md](../db/README.md)）。
- **变更与能力定义**：`openspec/changes/` 记录每次变更过程，`openspec/specs/` 记录能力级长期定义。
