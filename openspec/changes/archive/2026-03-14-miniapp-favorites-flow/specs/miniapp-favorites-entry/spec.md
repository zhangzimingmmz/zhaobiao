## ADDED Requirements

### Requirement: Favorites can be opened from the home page
The miniapp SHALL expose a working favorites entry from the home header.

#### Scenario: User taps favorites in the home header
- **WHEN** the user taps the favorites action in the home page header
- **THEN** the miniapp navigates to the favorites page

### Requirement: Favorites can be opened from the profile page
The miniapp SHALL expose a working favorites entry from the profile page.

#### Scenario: User taps favorites in profile
- **WHEN** the user taps the favorites menu item in the profile page
- **THEN** the miniapp navigates to the favorites page
